Project Ideas

Overview

⬇

Nowadays smartphones always by our side, but we often use it for social media and entertainment, why not we use it to help students better manage their study by using different types of application as a study aid. My project will be developing a mobile all-in-one study application mainly for students, teachers and parents. It mainly uses to track down all the courses, classes and assignments in school or university for students. For teachers, it also helps to manage with different courses of different grades & classes. Not only that, parents can use the application if they would like to take care or keep track of their kid's study status.

Motivation

⬇

The study application is really interesting and useful, it involved various of powerful features with user-friendly interface to help students manage their class time and assignment due date, make students learning life easier and efficiency. After I have done some research, nowadays actually study application is really common for students.

Based on Education Technology (2017), " 87% of 1,294 students surveyed said that they use apps to help them study. Of the majority, the primary reason for using apps was to remember deadlines as that reduces stress dramatically and allows them to plan work accordingly."

__________

According to Education Technology (2017), “94% of the UK population owning smartphones.”. Majority of the people have smartphones, so for most of the people, there is no point not using a study application.

__________

Description

⬇

The features of the study app include helps the student to organize their class time, exam date & time, users allow to input and manage their courses, classes and exams. All the inputted classes will be displayed in a week-view calendar. It improves users time management skill. It also helps users to manage assignment due date, like setting a reminder for their homework or any tasks. Users can either organize their assignments by date, by course or by priority or sort them by completed or uncompleted. All the inputted assignments will be displayed in the week-view calendar as well. The application has a powerful notification feature, whenever a user has inputted classes or assignments with due date, by default 24 hours before that, it will send a notification to the user, the time range can set by the user, so this is similar to a reminder, mainly used to make sure students will not miss their class and assignment.

Not only that, the application also has a simple note-taking feature, users can create and rename digital notebooks for a specific subject, create pages with title and allow users to change different paper style, type, write, highlight, draw and insert photos on pages, it also have document scanning feature, allow users to take photo on a document or any notes, crop the photos and auto enhancing make the texts or image look clearer and sharper, allow it to insert to a specific page of a digital notebook.

Other than that, users can register a free account for this app with a username, email address and password set to unlock some features, after users registered an account, they can manage their profile like update their name, school and profile picture. One of the locked features is gives users the ability to sync all the contents across different devices like class timetables, assignment due dates, notebooks and so on. All users need do is to register an account and sign in in different devices, users can view all the contents across different devices. The next locked feature is the GPS feature, use to search the nearby best study location like the nearby library. Last but not least, this application is available in iOS and Android both operating systems.
